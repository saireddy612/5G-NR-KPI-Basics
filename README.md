# 5G-NR-KPI-Basics
This project demonstrates a basic correlation between key 5G NR performance indicators:
**SINR, MCS, BLER and DL&UL Throughput**

## Objective
The objective is to understand how radio quality and scheduler decisions impact user throughput under different RF conditions.

## KPIs Analysed
- SINR
- MCS Index
- BLER
- Downlink Thorughpu
